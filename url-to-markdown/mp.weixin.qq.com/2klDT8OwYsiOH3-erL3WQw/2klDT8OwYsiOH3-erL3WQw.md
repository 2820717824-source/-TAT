# 从“养老”到“享老”：镁信健康与中信养老签署战略合作协议，携手共建健康养老新生态

*2026年6月9日 17:00*

![图片](https://mmbiz.qpic.cn/sz_mmbiz_gif/S5eRmGBYW26QibBlsXRNiaumia9DhJp5HNWPmSGsGGt4CjdEPLaVC9JSu81cWBFkQI3RjBIBQFx5CusbzHgiakRvqABjUWs3yibaQlibyFvibCUjAQ/640?wx_fmt=gif&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=0)

面对中国60岁以上老年人已达3.2亿的人口结构新格局，“十五五”规划建议明确提出大力发展银发经济，标志着我国应对人口老龄化的战略正式从被动保障转向主动布局。在这条高度确定、规模万亿级的银发经济黄金赛道中，养老产业已然告别传统单一生活照护模式，加速迈入“养老+医疗+保障+科技”深度融合的全新发展阶段。

近日，在2026上海国际养老、辅具及康复医疗博览会（以下简称“上海老博会”）现场，国内领先的医药多元支付平台镁信健康与中信集团专属养老服务平台中信养老正式达成战略合作， **完成了央企康养硬实力与民营科技创新力的深度牵手，标志着养老服务体系与健康保障服务体系进一步走向深度融合，为银发经济高质量发展提供实践样本。**

在长寿时代背景下，老年群体对医疗服务可及性、健康保障覆盖度以及疾病管理连续性的需求持续上升，传统养老服务已难以满足新时代长者的多元化需求。此次签约正是双方围绕“医、药、康、养、付”融合发展趋势展开的重要探索，实现了 **央企稳健产业基底、全域资源布局与民营科技研发能力、场景创新思维的深度绑定** 。

中信养老作为中信集团养老业务发展平台，中信兴业投资集团成员企业，现有投资规模超过30亿元，深耕以上海为核心的长三角地区，并开始布局北京、大湾区等重点城市。中信养老以优质的“养老运营管理+康养不动产管理”作为公司核心能力，专注于“市场化、可支付、高品质”的集中式养老服务产品，以“都市型、一站式、医康养”为特色，提供“养、护、医”不同类型的养老全周期解决方案。

镁信健康作为国内领先的医药多元支付领域的重要创新力量，近年来始终深耕医疗健康与支付保障赛道，持续构建开放型产业生态整合能力，目前已与数百家保险机构、药企以及头部医疗服务企业建立了深度战略合作，打造了覆盖药品、器械、服务的多元支付与创新服务网络，是推动商业健康险与创新医疗融合发展的核心平台。

随着越来越多优质资源向银发经济领域汇聚，一个覆盖全生命周期、更加智能高效、更具普惠价值的健康养老新生态正在加速形成。未来，双方将立足国家战略导向，聚焦新时代长者全周期健康养老核心需求，围绕老年群体健康保障、医疗服务、支付创新及养老场景服务升级等领域展开深度合作，共同探索覆盖养老全链条的一站式健康服务解决方案，助力中国老年人从“养老”迈向品质“享老”新阶段。
